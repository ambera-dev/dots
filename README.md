# Dotfiles
Dotfiles I use for Kali Linux.  
   
**Display Manager:** SDDM (Simple Desktop Display Manager)  
**Window Manager:** bspwm + polybar    
**Launcher:** rofi  
**Terminal:** st    
**Editor:** vim  
**Icon Theme:** Black Slate NUMIX Flat
